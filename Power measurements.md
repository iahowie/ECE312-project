2. __Ultrasonic Sensors (x2)__

   - For a single sensor:
   
      $V = 5.01 \ V$
   
      $I = 2.22 \ mA$
   
      $P_{2, single} = V * I = 5.01*2.22 = 11.1 \ mW$

   - For two sensors:

      $P_{2, total} = 2*P_{2, single} = 2 * 11.1 = 22.2 \ mW$
